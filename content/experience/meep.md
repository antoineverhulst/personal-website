## Data Analyst - Mobility as a Service

Led data infrastructure setup for a Mobility as a Service (MaaS) application, integrating multiple transportation modes into a unified analytics platform.

### Infrastructure Development

- **Database Integration:** Connected Firebase and PostgreSQL databases using Google BigQuery
- **Visualization:** Created dashboards in Data Studio and Power BI for multiple stakeholders
- **Architecture:** Designed scalable data infrastructure from the ground up

### Cross-Functional Collaboration

Worked closely with:
- Design team on user experience optimization
- Development team on data requirements and API design
- Management team on strategic KPIs and business metrics

### Key Deliverables

#### Multimodality Analytics
- Defined indicators to track user behavior across different transport modes (bike, scooter, public transport, etc.)
- Improved user experience through data-driven insights on multimodal journey patterns

#### Fleet Management
- Created visualizations and analyses for fleet operations
- Enabled better resource allocation and operational efficiency

### Impact

- Established the foundation for data-driven decision making
- Enabled real-time monitoring of multimodal transportation usage
- Provided actionable insights for improving user experience

### Skills Applied

SQL, BigQuery, Data Studio, Power BI, Firebase, PostgreSQL, Multimodal Transportation Analytics, Dashboard Design
